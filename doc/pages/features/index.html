Synery Query Language

Easily import, modify, link and export data within one language designed for data architects.

  • Well-thought-out mix between less syntax and good readability
  • Integrated import and export commands
  • Support for functional programming
  • Object compositions (records)
  • Type-safe
  • Event and exception handling
  • SQL-like commands

Extendable

Designed to extend, Interface Booster supports different types of plugins written in nativ .NET code.

  • Provider Plugins
    Used to import and export data from and to foreign systems or files.
  • Library Plugins
    Used to extend the functionallity of Synery by native .NET code.

Optimized for Speed

Interface Booster was designed with speed in mind. Therefore we implemented our own integrated database.

  • Integrated database
  • In-memory operations
  • Optimized for handling large amounts of data

What does the Interface Booster do?

Interface Booster is a software for fast developing stable and high quality interface applications which exchange business data between any application, format, file or service.

One of the big benefits of using Interface Booster is that the developer doesn't have to deal with the specifics of the foreign systems and formats. A Provider Plugin will do that for him.
