Execute draft

The EXECUTE-Command is used to export modified data to a foreign system or a file by using a connection (see: Connect).

The EXECUTE-Command

A EXECUTE-command always uses this pattern:
EXECUTE [endpoint_path] {SET-Command}? {GET-Command}? {Another-Command}* END

Description:

EXECUTE [endpoint_path]
Required. A path of an endpoint starting with two backslashes \\. The identifier starts with the name of the connection followed by the path to the endpoint. An example: \\myConnection\someCategory\endpointName.
{SET-Command}?
Optional. A SET-command containing some parameters.
{GET-Command}?
A GET-command requesting for some values to be returned.
{Another-Command}*
Optional. Some nested commands.
