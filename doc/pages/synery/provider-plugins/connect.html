Connect draft

The Connect-Command is used to create a connection to a foreign system or a file by using a provider plugin. A connection is required to perform import and export commands.

The Command

A connect-command always uses this pattern:
CONNECT "[provider_plugin_instance_name]" ( [parameters] )? AS [connection_identifier] ;

Description:

[provider_plugin_instance_name]
The syneryIdentifier of the plugin instance specified in the Interface Definition.
( [parameters] )?
A comma-separated key-value list of parameters which can/must be used to configure the connection.
connection_ identifier
An external identifier starting with two backslashes \\. An example: \\myCategory\someConnection.
