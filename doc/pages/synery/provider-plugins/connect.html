Connect draft

The Connect-Command is used to create a connection to a foreign system or a file by using a provider plugin. A connection is required to perform import and export commands.

The Command

A connect-command always uses this pattern:
CONNECT "[provider_plugin_instance_name]"
AS [connection_identifier]
SET ( [parameter_name] = [expression] , ... )
END

Description:

CONNECT "[provider_plugin_instance_name]"
Required. Takes the syneryIdentifier of the plugin instance specified in the Interface Definition.
AS [connection_identifier]
Required. Takes the external identifier starting with two backslashes \\. An example: \\myCategory\someConnection.
SET ( [parameter_name] = [expression] , ... )
Optional. The SET-argument is used to specify a comma-separated key-value parameters list that can/must be used to configure the connection.
