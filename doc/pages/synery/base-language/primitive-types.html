Primitive Data Types

Synery supports a few primitive data types that are based on the native .NET types (64bit). See: MSDN: Built-In Types Table (C# Reference).

Here is a list of supported types and a few examples how they can be used:

STRING

A string is used for texts. It represents a sequence of zero or more Unicode characters. It is written in double quotes (").

It is also possible to write a multi-line text:

INT

An integer number without decimal places. (See: MSDN: Int64)

DECIMAL

A floating point number for example used for financial and monetary calculations. (See: MSDN: decimal)

DOUBLE

A 64-bit floating-point number. (See: MSDN: double)

CHAR

One single Unicode character. (See: MSDN: char)

DATETIME

Represents a date and/or a time value. (See: MSDN: DateTime)
