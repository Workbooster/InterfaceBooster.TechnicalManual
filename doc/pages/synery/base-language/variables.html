Variables reviewed

Every variable in Synery has its type. This can be a primitve data type or a record type. Once a variable is declared its type cannot be changed. Synery is so-called "type safe".

Declaring a Variable

A variable is declared by defining a type and a name:

At this time the variable isn't initialized with a value yet. Its current value is NULL.

The variable name must start with a letter or an underscore _.

A few examples for possible variable names:

  • myVariable01
  • MyVariable01
  • _my3Mixed524Variable

A few examples for invalid variable names:

  • 1variable
  • +my-variable
  • variable+With-Special/\Chars$

Initializing a Variable

A variable is initialized by setting a value using the equality sign "=".
