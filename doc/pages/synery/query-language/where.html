Where draft

Like in SQL the WHERE command is used to filter the result set. But unlike in SQL the position of the WHERE command doesn't matter and there can also be multiple WHERE commands per request.

An Example

Assume we have a table like this:

\people
Id (INT) IsMale (BOOL) Firstname (STRING) Lastname (STRING) Size (INT)
15 TRUE Mike Meyer 182
22 FALSE Sandra Smith 168
27 TRUE Peter Muller 169
38 FALSE Rachel Lopez 175

Now let's have a look how a Synery query could look like when querying the table from above:

On the third line you see a WHERE command that filters for people taller than 170cm.

On the fourth line we have a SELECT command that defines the structure of the resulting table.

On the sixth line we have again a WHERE command that filters the current result set for people that are male.

On the last line we have again a SELECT command that finally defines the structure of the resulting table.

The result will look like this:

\myTables\TallMalePeople
FullName (STRING) Size (INT)
Mike Meyer 182
