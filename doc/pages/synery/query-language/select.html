Select draft

Like in SQL the SELECT statement is used to chose and restructure the requested fields.

An Example

Assume we have a table like this:

\people
Id (INT) Firstname (STRING) Lastname (STRING) Size (INT)
15 Mike Meyer 182
22 Sandra Smith 168
38 Rachel Lopez 175

Now let's have a look how a Synery query could look like when querying the table from above:

On the first line you see a table statement with an equality sign =. This mean that the result of the following query should be stored in the table with the given name. In this case a new table with the name \myTables\bigPeople will be created.

On the second line we have a FROM statement that declares which is the source table (in this case \people) an what alias (here: p) should be used for that table.

On the third line we have a SELECT statement that defines the structure of the resulting table.

The result will look like this:

\myTables\bigPeople
FullName (STRING) Size (INT)
Mike Meyer 182
Sandra Smith 168
Rachel 175
