Welcome to this Manual

This is the technical manual of the Interface Booster environment and its scripting language "Synery". It can be used as reference for developers and technicians.

Who is this manual for?

This manual is for users who want to use Interface Booster to create interface applications.

Terms you should know:

Synery
Synery is a scripting language that is used to implement the business logic of an interface application. See: Synery Language
Provider Plugin
A plugin used to exchange data with a foreign system or a file. For example there could be a plugin that allows to communicate with a webservice of an external application. See: Provider Plugins
Library Plugin
A plugin used to extend the functionality of Synery. For example there could be a plugin that provides functions to convert a DATETIME value into a STRING. See: Library Plugins
Interface Definition
Consist of multiple files that together with the Interface Booster Runtime software build the interface application.
Interface Booster Runtime Environment
The Software (.EXE) used to execute one or multiple jobs of an interface definition.
