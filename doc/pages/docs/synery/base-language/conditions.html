Conditions draft

There are two types of conditions:

If-Else Condition

Here you see a simple condition that checks whether the value is greater than ten. Because it isn't nothing happens.

Here you see a condition with an ELSE-block that checks whether the value is greater than ten. The ELSE-block is executed.

Here you see a condition with multiple ELSE IF-blocks and an ELSE-block that makes some checks on the value. The second ELSE IF-block is executed.

Ternary Operation

Synery supports ternary operations. Ternary operations are used to conditionally set a value.

They are written in the form: [Condition] ? [True-Value] : [False-Value]
