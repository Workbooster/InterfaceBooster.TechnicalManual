Interface Definition

The Interface Definition consist of multiple files that together with the Interface Booster Runtime software build the interface application.

File System Structure

importDefinition.xml

This XML file contains these information:

  • Information about the interface application for the end user.
  • Technical information about the required environment.
  • Plugin references
    • A list of all required Provider Plugins.
    • A list of all required Library Plugins.
  • Interface Jobs
    • Information about the jobs for the end user.
    • A list of Synery code files which should be included to the main job Synery code file during execution.
