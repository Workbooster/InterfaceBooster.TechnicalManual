Order By draft

Like in SQL the ORDER BY command is used to sort the data in the result set.

One query statement can contain multiple ORDER BY commands. Each command is based on the result set of the prior commands.

An Example

Assume we have a table like this:

\people
Id (INT) IsMale (BOOL) Firstname (STRING) Lastname (STRING) Size (INT)
15 TRUE Mike Meyer 182
22 FALSE Sandra Smith 168
27 TRUE Peter Muller 169
38 FALSE Rachel Lopez 175

Now let's have a look how a Synery query could look like when querying the table from above:

On the third line we have a ORDER BY command that defines the sorting of the resulting table. In this case we'd like to have the tallest person on top and the shortest person on the bottom of the table. For that reason we use the keyword DESC to get a descending order.

The result will look like this:

\myTables\sortedPeople
Id (INT) IsMale (BOOL) Firstname (STRING) Lastname (STRING) Size (INT)
15 TRUE Mike Meyer 182
38 FALSE Rachel Lopez 175
27 TRUE Peter Muller 169
22 FALSE Sandra Smith 168
