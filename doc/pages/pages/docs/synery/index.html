Synery Language

Synery is a scripting language that is used to implement the business logic of an interface application.

Features

  • Well-thought-out mix between less syntax and good readability
  • Integrated import and export commands
  • Support for functional programming
  • Object compositions (records)
  • Type-safe
  • Event and exception handling
  • Expandable with native .NET code (Library Plugins)
  • Powerful query language
    • SQL-like commands
    • In-memory operation
    • Optimized for handling large amounts of data
  • Integrated database
    • Optimized for speed
    • Well integrated with the query language

Comparing to Other Languages

Synery is most similar to SQL. But Synery is not just a query language. In much cases it behaves different than SQL. For example in Synary it is possible to define and use functions directly inside of a code file. There is no need to store the functions first as some kind of a database object.
