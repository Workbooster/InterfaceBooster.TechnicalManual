Questions and Anwers

A Question is used to specify a parameter. An Answer is the containers for the parameter value.

The Provider Plugin can "ask" the interface developer a few questions. Answering a questions can be optional or required.

Interface Booster checks whether all required questens are answered and whether the data types match.
