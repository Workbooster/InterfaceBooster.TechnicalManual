Create modern software interfaces

Interface Booster is a software for fast developing stable and high quality interface applications which exchange business data between any application, format, file or service.

Easy to learn

The Synery script language, used to write interface applications, is similar to SQL.

Open Source

The Runtime Environment is free software licensed under the terms of the GNU Lesser General Public License (LGPL).

Extensible

Interface Booster can easily be extended becuase of its plugin architecture.

Maintainable

Interfaces are easy to maintain because there is no compiler or build engine needed.

Powerful query language

Optimized for working with business data Interface Booster lets you focus on your data.

Speed

An integrated database in combination with in-memory operations guarantees a fast data-flow.

Made for data architects

Focus on your business logic, not on technical details.
