Endpoints

A Provider Plugin defines its' operations by providing endpoints. Each data command runs against an endpoint.

What endpoints are available and what commands are allowed is specified by the Provider Plugin.

The endpoint architecture of the Provider Plugin API is oriented on the architecture of RESTful Webservices.

Base Interface

The ProviderConnection contains a list of all available endpoints. Each endpoint must implement the interface IEndpoint.
